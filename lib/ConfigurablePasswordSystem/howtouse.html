The Configurable Password System

This Password System Is a Nice, Lightweight, Simple Configurable password system written in C++.
The Password System works like every other, Specify Default passwords, Add attached passwords to accounts, And Just do more.
This Password System may be one of the first to include

  • Configurations
  • Simple Data Gathering
  • Source Code
  • Unix-like Password Management
  • Access Denial & Giving
